# LibraryProject

This is a basic Django project created for learning purposes.

## Structure
- `manage.py`: Command-line tool for project tasks.
- `settings.py`: Configuration file.
- `urls.py`: URL routing.
- `wsgi.py` / `asgi.py`: Entry points for servers.

## How to Run
1. Install dependencies: `pip install django`
2. Run server: `python manage.py runserver`
3. Visit: `http://127.0.0.1:8000/`
